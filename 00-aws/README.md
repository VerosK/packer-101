
# AWS preflight check

## AWS credentials preparation

1. Get your `aws_access_key_id` and `aws_access_key_id` and put them
   to `aws_credentials.ini`

2. Export `AWS_PROFILE` and `AWS_SHARED_CREDENTIALS_FILE` environment variables.
